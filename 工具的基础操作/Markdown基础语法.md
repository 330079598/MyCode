# typora快捷键

- 表格

  - 可以用快捷键 **Ctrl+t** 
  - **|空格|空格|**，接着**enter**会弹出来表格

- 文字背景颜色

  - 打开**高亮功能**，`==内容==`，下面是演示
  - ==文本内容==

  ```markd
  | 左对齐标题 | 右对齐标题 | 居中对齐标题 |
  | :------| ------: | :------: |
  | 短文本 | 中等文本 | 稍微长一点的文本 |
  | 稍微长一点的文本 | 短文本 | 中等文本 |
  ```
