# 分拆素数和

**Time Limit: 1000/1000 MS (Java/Others)    Memory Limit: 32768/32768 K (Java/Others)Total Submission(s): 52559    Accepted Submission(s): 23123**

Problem Description

把一个偶数拆成两个不同素数的和，有几种拆法呢？

 

Input

输入包含一些正的偶数，其值不会超过10000，个数不会超过500，若遇0，则结束。

 

Output

对应每个偶数，输出其拆成不同素数的个数，每个结果占一行。

 

Sample Input

```
30
26
0
```

 

Sample Output

```
3
2
```

- 所得的个数要去重

```c
#include<iostream>
#include<cmath>
using namespace std;

int su(int n)
{
    int m = sqrt(n);
    for(int j=2;j<=m;j++)
            {
                if(n%j == 0)
                    return 0;
            }
            return 1;
}

int main(void){
	int sum=0;
	int n;
	while(cin>>n,n != 0)
	{
		sum=0;
		for(int j = 2;j < (n+1)/2;j++)
		{
			if(su(j))
			{
				if(su(n-j))
					sum++;
			}
		}
		cout<<sum<<endl;
	}
	return 0;
}
```
