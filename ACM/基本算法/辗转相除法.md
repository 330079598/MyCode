# 辗转相除法

- 作用：
  - 可以用来求最大公约数
  - 可以求两数的最小公倍数
- 原理：若a除以b的余数为r，则有（a，b）=（b，r），递归后，b就是他的最大公约数。
- 算法演示：
- 方法一：

```c++
int gcd(int m,int n)
{
        int t = 1;
        while(t != 0)
        {
                t=m%n;
                m=n;
                n=t;
        }
        return m;
}
```

  - 方法二：

```c++
int gcd(int m,int n)
```

- 方法三：递归

```c++

#include <iostream>
#include <conio.h>
using namespace std;
 
int gcd(int a, int b){
	return b == 0 ? a : gcd(b, a % b);
}

{
	cout << gcd(169, 48) << endl;
```
