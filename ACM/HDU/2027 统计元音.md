# 统计元音

**Time Limit: 2000/1000 MS (Java/Others)    Memory Limit: 65536/32768 K (Java/Others)Total Submission(s): 105693    Accepted Submission(s): 40448**

Problem Description

统计每个元音字母在字符串中出现的次数。

 

Input

输入数据首先包括一个整数n，表示测试实例的个数，然后是n行长度不超过100的字符串。

 

Output

对于每个测试实例输出5行，格式如下：
a:num1
e:num2
i:num3
o:num4
u:num5
多个测试实例之间由一个空行隔开。

请特别注意：最后一块输出后面没有空行：）

 

Sample Input

```tcl
2
aeiou
my name is ignatius
```



Sample Output

```txt
a:1
e:1
i:1
o:1
u:1

a:2
e:1
i:3
o:0
u:1
```

```c++
#include<iostream>
#include<string>
using namespace std;

int main(void)
{
	int n,j;
	string s;
	cin>>n;
	getchar();
	while(n--)
	{
	//getchar();
     //诺，就是这里，如果getchar放在这个，每个while都会吃一个回车，答案就错了，以后一定要想的周全，认真
		getline(cin,s);
		int len = s.size();
		int a = 0;
		int e = 0;
		int i = 0;
		int o = 0;
		int u = 0;
		for(j = 0;j<len;j++)
		{
			if(s[j] == 'a')
				a++;
			if(s[j] == 'e')
				e++;
			if(s[j] == 'i')
				i++;
			if(s[j] == 'o')
				o++;
			if(s[j] == 'u')
				u++;
		}
		cout<<"a:"<<a<<endl;
		cout<<"e:"<<e<<endl;
		cout<<"i:"<<i<<endl;
		cout<<"o:"<<o<<endl;
		cout<<"u:"<<u<<endl;
		if(n)
			cout<<endl;
	}
	return 0;
}
```
